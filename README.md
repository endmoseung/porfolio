## 실행 방법

레포지토리를 `clone` 합니다
```markdown
$ git clone https://github.com/Pre-HotSix/wanted-pre-onboarding-imlab.git
```
dependencies를 설치합니다
```markdown
$ npm install
```
프로젝트를 실행합니다
```markdown
$ npm start
```
<br/>

## 프로젝트소개
1. HTml,Css,Javascript파일로만 만든 프로젝트
2. Css는 Vem기법으로 작성
